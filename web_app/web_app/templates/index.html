MachineLurgy


About

MachineLurgy is a web-based application that can predict the mechanical properties of low-alloy steel when provided with required compositons. As currently there are no precise theoretical methods to predict mechanical properties of steels, all the methods available are backed by statistics and extensive physical testing of the materials.

Since testing each material with different composition is a highly tedious task (imagine the number of possibilities!), MachineLurgy helps practitioners to come up with a set of best low-alloy steels that possess required mechanical properties just by providing composition of the steel.
